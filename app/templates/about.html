About this Project

This project was part of my MSc Data Science programme at Lancaster University. This website is actually just an interactive way of displaying the results of an extensive, targetted analysis of the UK MOT vehicle testing data set. It is aimed at providing information on the pass rates, top faults discovered at MOT test failures, and the actual detailed descriptions of these faults.

The project can be divided up into three main stages - data integration, analysis and presentation. The data collection was basically hours and hours of downloading, inspecting and concatenating of the data from the official UK MOT vehicle testing data website, part of the UK government's open data. THe three stages are explained in more detail below

Data Integration

The first main step in this project was data integration. As the documentation for the data wasn't so clear, much of the ideas for the data integration came only after much playing around with the data, inspection and manipulation of the data using command line tools, and in some cases, loading small chunks up into databases and running SQL scripts on them (call it trial and error, if you like).
